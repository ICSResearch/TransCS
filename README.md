# TransCS
This repo is the pytorch code for paper "TransCS: A Transformer-based Hybrid Architecture for Image Compressed Sensing".  
We built and tested with an Intel Xeon Silver 4210 CPU and a GeForce RTX 3090 GPU.
****
## _Requirements_
Python 3.9  
CUDA 11.1  
PyTorch 1.9.0  
torchvision 0.9.0  
****
